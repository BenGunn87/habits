Упражнение

Мягкая мобилизация утром

Короткая последовательность для шеи, плеч и грудного отдела, которая помогает мягко включиться в день.

← Вернуться к каталогу упражнений

В стартовой версии подключен очень легкий локальный MP4-файл для демонстрации структуры GitHub Pages. Его можно заменить собственным роликом без изменения страниц.

Упражнения

  1. Сядьте или встаньте устойчиво, опустите плечи и сделайте три спокойных вдоха.
  2. Выполните медленные повороты головы вправо и влево без рывков.
  3. Сделайте круги плечами назад, затем сведите и мягко раскройте лопатки.
  4. Добавьте несколько плавных раскрытий грудной клетки с опорой на дыхание.
