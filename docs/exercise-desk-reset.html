Упражнение

Разгрузка после долгого сидения

Набор мягких движений для перерыва между рабочими блоками: раскрытие передней линии тела и включение ног.

← Вернуться к каталогу упражнений

В текущей сборке используется локальный demo-клип малого размера. Он нужен, чтобы раздел видео был готов к публикации на GitHub Pages уже сейчас.

Упражнения

  1. Встаньте, поставьте стопы на ширину таза и несколько раз мягко перенесите вес с пятки на носок.
  2. Сделайте легкий выпад назад с опорой руками на бедра или стол.
  3. Добавьте вытяжение передней поверхности бедра и несколько плавных наклонов таза.
  4. Завершите упражнение спокойной ходьбой по комнате в течение 30-40 секунд.
